# 小说写作助手

专业的小说创作支持，涵盖构思、规划、写作到修改的全流程。

---

## When to Use

当用户提出以下需求时使用本技能：
- 从想法或类型出发，新建小说
- 基于已有章节进行续写
- 设计角色（主角、反派、配角）与角色弧线
- 构建世界观（势力、规则、魔法体系、时代背景）
- 对已有内容做润色、重写或节奏优化

## Do not use

以下场景不应使用本技能：
- 非创作类事实问答（如百科、实时资讯查询）
- 法律、医疗、金融等专业建议
- 抄袭或要求复制受版权保护作品风格的请求
- 违反平台安全规则的有害内容请求

## Instructions

1. 先判断任务类型（新建、续写、角色、世界观、修改）。
2. 先问最少必要信息，再开始输出。
3. 按本文件对应流程章节执行。
4. 优先复用 `references/` 与 `assets/templates/` 中的材料。
5. 保持视角、语气、角色动机与上下文一致。
6. 输出前做一次情节连贯性与语言质量自检。

## 新建小说

### 创作流程

```
1. 沟通基础信息 → 2. 构建框架 → 3. 角色设计 → 4. 章节规划 → 5. 开始写作
```

### 1. 沟通基础信息

询问用户以下关键信息（不要一次问太多）：

**必问：**
- 小说类型（科幻/奇幻/悬疑/言情/武侠等）
- 故事核心概念或灵感
- 目标篇幅（短篇/中篇/长篇）

**可选（可逐步询问）：**
- 主角设定
- 想要的情感基调
- 特殊要求

### 2. 构建框架

根据收集的信息，使用**三幕结构**创建故事大纲：

- **第一幕（25%）**：建置 → 激励事件 → 踏上旅程
- **第二幕（50%）**：挑战 → 中点转折 → 危机 → 准备
- **第三幕（25%）**：高潮 → 解决 → 新常态

**详细结构指导**：参见 [references/story-structure.md](references/story-structure.md)

### 3. 角色设计

为核心角色创建角色卡：

- **主角**：核心欲望、核心恐惧、角色弧线
- **反派**：动机、与主角的冲突
- **关键配角**：功能定位

**详细角色指导**：参见 [references/character-development.md](references/character-development.md)

### 4. 章节规划

根据大纲规划具体章节，每章明确：
- POV（视角）
- 场景地点
- 情节要点
- 与整体故事的连接

### 5. 开始写作

按照规划逐章写作，注意：
- 保持视角一致
- 场景细节生动
- 对话推动情节
- 适当埋设伏笔

## 续写章节

### 续写前准备

1. **理解现有内容**
   - 阅读已有章节
   - 梳理当前情节进展
   - 确认角色状态和关系

2. **确认续写方向**
   - 下一章的目标是什么？
   - 需要发生什么事件？
   - 情感走向如何？

### 续写要点

- 承接上文情节和氛围
- 保持角色性格一致
- 推进主线或支线
- 结尾留有钩子

---

## 角色设计

### 设计流程

1. **确定角色功能**
   - 主角/反派/配角？
   - 在故事中的作用？

2. **填充角色维度**
   - 基础信息（姓名、年龄、外貌）
   - 核心欲望和恐惧
   - 优点和缺点
   - 独特标识（说话方式、习惯动作）

3. **设计角色弧线**
   - 开始状态
   - 经历的变化
   - 结束状态

4. **建立关系网**
   - 与其他角色的关系
   - 冲突和羁绊

**使用模板**：[assets/templates/character-card.md](assets/templates/character-card.md)

## 世界观构建

### 构建层次

1. **物理世界**
   - 地理环境
   - 气候生态
   - 重要地点

2. **社会体系**
   - 政治结构
   - 经济系统
   - 文化习俗

3. **特殊系统**
   - 魔法体系（奇幻）
   - 科技设定（科幻）
   - 武功系统（武侠）

4. **历史背景**
   - 重要事件
   - 传说神话

### 构建原则

- **服务故事**：只构建故事需要的内容
- **内部一致**：规则不自相矛盾
- **展示而非讲述**：通过情节展现世界

**详细指导**：参见 [references/worldbuilding.md](references/worldbuilding.md)

## 写作技巧

关键技巧速览：

- **叙事视角**：第一人称/第三人称限知/全知，根据故事需要选择
- **场景构建**：时间+地点+人物+冲突+变化
- **对话写作**：推动情节、展现性格、有潜台词
- **节奏控制**：张弛有度，高潮后需缓冲
- **悬念伏笔**：提前埋设、后期回收

**详细技巧**：参见 [references/writing-techniques.md](references/writing-techniques.md)

## 输出规范

### 大纲输出
- 结构清晰，层次分明
- 关键节点明确
- 便于后续扩展

### 正文输出
- 中文写作，使用中文标点
- 段落适当分段
- 对话格式规范
- 避免过于书面化，保持可读性

### 格式示例
```
第一章 暗夜降临

城市的霓虹灯在雨幕中模糊成一片光晕。林远站在天台边缘，任由冰冷的雨水打湿他的风衣。

"你来了。"身后传来一个熟悉的声音。

他转过身，看到一个黑影正从楼梯口走来。
```
