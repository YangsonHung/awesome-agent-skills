## 角色原型

### 经典原型
- **英雄**：故事核心，推动情节
- **导师**：引导英雄，提供智慧
- **守门人**：测试英雄的决心
- **信使**：带来改变的消息
- **变形者**：立场模糊，增添悬念
- **阴影**：反派，代表主角的恐惧
- **盟友**：支持主角，提供帮助
- **捣蛋鬼**：带来混乱和幽默

## 角色维度

### 基础信息
- 姓名、年龄、外貌
- 职业、社会地位
- 家庭背景

### 内在维度
- **核心欲望**：角色最想要什么？
- **核心恐惧**：角色最害怕什么？
- **核心谎言**：角色相信的错误信念
- **核心真相**：角色需要学会的真理

### 外在维度
- **公开目标**：角色表面上追求的
- **隐秘动机**：深层的驱动力
- **冲突来源**：内外在的障碍

### 独特标识
- 独特的说话方式
- 标志性动作或习惯
- 特殊技能或弱点

## 角色弧线

### 积极弧线（成长型）
角色克服内在缺陷，变得更好。
1. 以缺陷开始
2. 面对挑战
3. 挣扎与失败
4. 觉醒时刻
5. 胜利与转变

### 消极弧线（堕落型）
角色屈服于缺陷，变得更糟。
1. 以希望开始
2. 面对诱惑
3. 做出错误选择
4. 螺旋式下降
5. 悲剧结局

### 平坦弧线（不变型）
角色改变世界，自己保持不变。
1. 以真理开始
2. 面对谎言的世界
3. 坚持信念
4. 影响他人
5. 胜利

## 角色关系网

### 主要关系类型
- **对立**：主角 vs 反派
- **浪漫**：爱情线
- **友谊**：支持系统
- **家庭**：血缘羁绊
- **师徒**：指导关系
- **竞争**：势均力敌

### 关系动态
- 关系如何随故事发展变化
- 冲突如何影响关系
- 关系如何推动情节
